SafeSight

SafeSight is a GenAI-enhanced, Edge AI-powered safety monitoring system that processes live or uploaded video feeds directly on Copilot+ PCs powered by the Snapdragon X Elite processor. Supervisors define restricted and safe zones in the camera view, and the system automatically detects workers, verifies if they're wearing the correct PPE, and issues real-time alerts (buzzers, blinking lights) for violations. It also uses visual re-identification to monitor individual work durations and ensure shift-time compliance. A web-based manager portal allows natural-language queries like “How many workers were present this week?” or “Show me a safety violation report,” powered by lightweight GenAI models. Core technologies include YOLOv8 for object and PPE detection and OpenCV with Python for video processing. The frontend portal is built using Flask and lightweight React and alerting integrates with IoT devices. By combining GenAI and Edge AI, SafeSight delivers real-time situational awareness, proactive safety enforcement, and data-rich insights. It’s a scalable, real-world solution that directly aligns with the hackathon’s goal.
